# Product Workflows

## 1. Household setup workflow

1. Create household profile.
2. Add users.
3. Add core accounts (Joint, Bills, Credit Card, Savings).
4. Mark primary income target account.

## 2. Income setup workflow

1. Add each income source.
2. Assign amount and frequency.
3. Assign destination account.
4. Verify normalized totals in fortnightly, monthly, and annual views.

## 3. Expense and debt setup workflow

1. Add all recurring expenses with category and frequency.
2. Set due pattern and payment account.
3. Mark debt expenses and attach debt details.
4. Mark bumpable items where deferral is acceptable.

## 4. Pay-cycle planning workflow

1. Open or create current pay cycle.
2. Review total income and required amount.
3. Work through checklist groups:
   - Must-pay
   - Smoothing contributions
   - Debt payments
   - Savings contributions
   - Discretionary items
4. Mark item statuses (Paid, Bumped, Skipped).
5. Enter actual payment amounts where needed.
6. Confirm transfer recommendations from Joint account.

## 5. Lookahead workflow

1. View upcoming pay cycles.
2. Identify heavy cycles and expected deficits.
3. Adjust bumpable items or savings contributions.
4. Re-check projected transfer plan.

## 6. Debt review workflow

1. Review each debt balance and progress.
2. Compare min vs target payment effects.
3. Check projected debt-free horizon.
4. Prioritize 0% debt closures for quick wins where useful.

## 7. Reconciliation workflow (optional)

1. Upload transaction CSV.
2. Map imported rows to accounts and categories.
3. Validate anomalies.
4. Adjust pay-cycle item actuals.
