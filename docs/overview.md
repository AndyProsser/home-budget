# Project Overview

## Purpose

Home Budget is a browser-based household budgeting and cashflow planning app designed for a two-person household with mixed payment cycles.

The app is built around one key reality: income may arrive fortnightly, while bills and obligations can be fortnightly, monthly, annual, or irregular.

## Primary goals

- Plan spending and transfers per pay cycle (fortnightly).
- Normalize all recurring costs into comparable cashflow views.
- Track shared accounts and planned money movement.
- Track debt payoff progress, including 0% debt deals.
- Provide a practical workflow for two users operating one household budget.

## Current scope

- One household per deployment.
- Two or more users can share the same household budget data.
- Manual data entry is the default.
- Optional CSV import is supported for transaction reconciliation.

## Out of scope for now

- Live bank integrations.
- Multi-household tenancy.
- Advanced debt optimization strategy automation.

## Product pillars

- Planning first: every pay cycle should produce a clear checklist.
- Visibility: users should quickly see surplus, deficit, and bill spikes.
- Explainability: all calculations should be deterministic and testable.
- Practical UX: clear terms, low clutter, and fast updates.
